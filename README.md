# OOPJAVALAB
Project Name: Java Calculator Application
Project Description:

This project is a menu-driven calculator application developed in Java that demonstrates core programming concepts learned over multiple weeks of coursework. The application allows users to perform arithmetic operations, store values in memory, and handle invalid input safely through exception handling. The goal of this project was to integrate object-oriented programming, control structures, collections, and error handling into a single cohesive program.

Project Tasks
* Task 1: Develop basic calculator functionality
Implement addition and subtraction using object-oriented programming
Create classes and methods to perform mathematical operations

* Task 2: Add logical operations and menu system
Implement multiplication and division operations
Design a user-friendly menu to allow repeated calculations

* Task 3: Implement calculator memory features
Add single-value memory storage
Implement a collection-based memory system using an ArrayList
Provide operations to view, remove, sum, and average stored values

*Task 4: Add exception handling
Prevent program crashes from invalid user input
Handle division by zero and non-numeric entries safely

* Task 5: Integrate all features into one application
Combine weekly assignments into a unified calculator program
Refactor code into separate classes for better organization

* Task 6: Document and publish the project
Add comments and JavaDoc headers to all classes and methods
Upload the final project to GitHub with proper documentation

Project Skills Learned
* Object-Oriented Programming (classes, methods, and encapsulation)
* Control structures such as loops and switch statements
* Exception handling using try/catch blocks
* Working with collections using ArrayList
* Writing clean, readable, and well-documented code
* Using GitHub to manage and publish code projects

Language Used
Java: Chosen because it reinforces strong typing, object-oriented design, and structured exception handling, making it ideal for learning software development fundamentals.
Development Process Used

Incremental Development: The project was built in weekly stages, with each phase adding new functionality such as logic operations, memory features, and exception handling. This allowed for continuous testing and improvement before integrating everything into a final application.
